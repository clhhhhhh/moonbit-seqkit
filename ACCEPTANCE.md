# Acceptance checklist

- [x] MoonBit is the only implementation language.
- [x] Public GitHub/GitLink mirrors can be attached before submission.
- [x] README, MIT license, example, CI, and tests are present.
- [ ] Replace the placeholder repository URL in `moon.mod`.
- [ ] Run `moon login` and publish with `moon publish`.
- [ ] Import the GitHub repository into GitLink and verify the default branch.

The last three items require the contributor's accounts and cannot be completed offline.
