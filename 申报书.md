## 2. 项目简介

MoonBit SeqKit 是一个以 MoonBit 为主要实现语言的轻量级数值序列分析库，面向命令行工具、教学实验、数据预处理和轻量遥测聚合等场景。项目坚持零第三方依赖，提供清晰、可复现、适合国产基础软件生态的序列处理 API，降低在 MoonBit 中进行常见数据分析和算法验证的门槛。

## 3. 项目方向与适用场景

项目属于 MoonBit 基础库和开发者工具方向，适用于：序列统计、滑动窗口分析、差分与前缀计算、排序检索、简单数值算法教学、CLI 数据处理，以及小规模遥测数据的聚合分析。

## 4. 拟实现的核心功能与交付物

- 基础统计：求和、乘积、最值、均值、中位数、方差、范围和分桶统计；
- 序列算法：前缀和、相邻差分、点积、稳定插入排序、线性搜索和二分搜索；
- 滑动窗口：窗口求和、均值、最值、范围和幂次聚合；
- 工程能力：MoonBit 模块化 API、命令行最小示例、完整核心测试、MIT 许可证和 GitHub Actions 持续集成；
- 发布交付：公开 GitHub/GitLink 仓库、README 安装与使用说明，以及 Mooncakes `clhhhhhh/moonbit-seqkit@0.1.0` 包。
